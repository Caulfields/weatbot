Рекламные креативы для Telegram-посевов

Задача

Продвигать Telegram-каналы в нише онлайн-образования и бизнес-курсов через посевы — размещение рекламных постов на сторонних каналах. Для каждого запуска нужно было написать несколько версий поста для тестирования.

Что я делал

Перед написанием анализировал канал клиента и канал, где будет размещен пост: тематику, tone of voice, аудиторию, продукт который продвигается. Использовал TGStat. Это важно — пост должен органично смотреться на чужом канале и при этом точно попадать в боль целевой аудитории.

Для каждого запуска писал около 5 версий поста с разными заходами: через боль, через результат, через сомнение, через авторитет эксперта, через срочность и т. д. Каждая версия — отдельный оффер.

Менеджер по закупкам выбирал каналы для размещения — я сосредотачивался на текстах. После тестирования смотрел на результаты и при необходимости дорабатывал формулировки.

Средний CTR креативов в нише онлайн-образования удерживал на уровне выше среднего по нише (1,5–3%).

Увидеть примеры рекламных постов можно в Telegram-портфолио.
